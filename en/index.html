Welcome to St2Py

 

Published 2015-11-11

Welcome to St2Py! St2Py is a team that focus on Network Security. Now, we have two main projects, BitCrypt and SecIptun.

BitCrypt Project

BitCrypt is an open source army level file encryption/decryption software. It’s easy to install and use, supports Linux, Windows, OSX, FreeBSD four main operating systems. Welcom everyone to use it, and give us advise.

SecIptun Project

SecIptun is an open source secure tunnel software to ensure network communication security. It also uses the army level encryption technologies. Welcom everyone to use it, and give us advise.
