BitCrypt Project

 

Published 2015-11-19

BitCrypt is an open source file encryption/decryption software, it has the following features:

  • Released under The MIT License, fully open source and completely free
  • Developed by Golang, safe, efficient, easy to deploy
  • Support Linux, Windows, OSX, FreeBSD four main operating systems
  • Using RSA + AES ciphers, the army level file encryption technologies
  • Support RSA 1024, 2048, 4096 key length
  • Support AES 128, 192, 256 key length
  • Support CFB, CTR, OFB iteration mode
  • Public key used for encryption, it can be shared with others
  • Private key used for decryption, it can decrypt the shared encrypted files
  • Encryption/decryption process can be separated, facilitate authority management

Generate RSA key pair

BitCrypt will generate public.pem and private.pem files. Public key is in public.pem file, private key is in private.pem file.
Note: Please backup the key files carefully, the encrypted files cannot be decrypted if the key files are lost!

How it works

The simplified BitCrypt encryption process is as follows:

  • Generate random AES key and AES initial vector
  • Encrypt key and iv by using RSA, write to encryption file
  • Encrypt file by using AES, write to encryption file

In general, BitCrypt using AES to encrypt files, and using RSA to encrypt AES key and initial vector.

The simplified BitCrypt decryption process is as follows:

  • Read encryption AES key and iv from encryption file
  • Decrypt key and iv by using RSA, obtain AES key and iv
  • Decrypt file by using AES, write to decryption file

Of coure, file hash will be calculated after decryption process. It can check if the decryption file is broken.
